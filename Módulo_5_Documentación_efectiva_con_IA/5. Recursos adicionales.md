## Plataformas SaaS de documentación (para referencia)

Mencionadas en la sesión pero fuera del scope del proyecto del máster por coste o enfoque:

| Plataforma |                              Precio | Cuándo tiene sentido                                                  |
| ---------- | ----------------------------------: | --------------------------------------------------------------------- |
| Mintlify   |                        Pro $250/mes | Equipos que publican API docs comerciales, DevRel, docs como producto |
| GitBook    | Premium $65/sitio/mes + $12/usuario | Knowledge bases mixtas (interno + externo), no solo código            |
| ReadMe.io  |                   Business $349/mes | Developer hubs colaborativos con escritores no técnicos + devs        |

Para el proyecto del máster y para la mayoría de proyectos internos: Starlight + GitHub Pages = $0 cubre el 90% de las necesidades.
